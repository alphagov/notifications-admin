Send files by email

Use double brackets to add a placeholder field to your template. This will contain a secure link to download the file. 

Download your file at: ((example::file))

Select ‘Attach files’ when you’re ready to send your email. Follow the instructions to upload your file.

If you upload more than one file, you need to identify each one in the placeholder.

If you use the Notify API, follow the instructions to send a file by email in the API documentation.
