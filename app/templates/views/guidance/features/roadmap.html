The GOV.UK Notify roadmap shows what we’re working on and some of the things we’ve done.

The roadmap is only a guide. It does not cover everything we do, and some things may change.

You can contact us if you have any questions about the roadmap, want to get involved in user research or have suggestions for new features.

Things we’re working on now

  • Launch a regular email about new features and improvements to GOV.UK Notify
  • Let services add QR codes to letter templates
  • Let services create letter templates that comply with the Welsh Language Act 1993
  • Integrate with our letter provider’s API to improve the letter sending process
  • Add more formatting options to email and letter templates
  • Continue to improve our front-end code to meet our targets for accessibility
  • Give users the option to set a custom data retention period that’s more than 7 days
  • Support services to write better notifications
  • Migrate our database from GOV.UK PaaS to one hosted using Amazon RDS
  • Provide more detailed reporting and usage data for services

Things we’ll do later

August to October 2023

  • Support services to write better notifications through reusable text message templates
  • Explore new channels for notifications
  • Explore ways to make it easier for services to send one-off messages
  • Migrate our applications from GOV.UK PaaS to Amazon ECS

Things we’ve done

  • Let services attach pages to letter templates
  • Give organisations permission to approve new services
  • Users can consent to take part in research
  • Introduce subheadings for emails
  • Make it easier to find information and guidance on the Notify website
  • Make it easier for services to request to go live (private beta)
