Send files by email

Notify offers a safe and reliable way to send files by email.

Upload a file using our API or the Notify website, then send users an email with a link to download it.

Notify uses encrypted links instead of email attachments because:

  • they’re more secure
  • email attachments are often marked as spam

Protect personal information about your users

To help protect your files you can:

  • ask recipients to confirm their email address before downloading
  • choose the length of time that a file is available to download

How to send a file by email

  1. Go to the Templates page.
  2. Add a new template or choose an existing template.
  3. Select Add files.
  4. Upload your file.
  5. Select Get ready to send

If you use the Notify API

Follow the ‘Send a file by email’ instructions in our API documentation. You’ll need a developer on your team to set it up for you.
