International text message limits

By default, you can only send 100 text messages a day to international phone numbers.

We set this limit because international text message fraud can be very costly.

If you need to change the daily limit, you should take steps to protect yourself from fraud.

For example:

  • work out how many international text messages you need to send each day and base your new limit on that
  • track how many messages you send and set an alert in your system for anything that is not normal
  • stop your service from repeatedly sending messages to the same number

For more suggestions, see the National Cyber Security Centre (NCSC) website.

How to change the international text message limit

  1. Go to {{ service_link(current_service, 'main.service_settings', 'Settings') }}.
  2. Select Change on the International text message limit row.

If you do not need to send any international text messages:

  1. Go to {{ service_link(current_service, 'main.service_settings', 'Settings') }}.
  2. Select Change on the Send international text messages row.
