Text message sender ID

The text message sender ID tells your users who the message is from.

To change the text message sender ID from the default of ‘GOVUK’:

  1. Go to the Text message settings section of the {{ service_link(current_service, 'main.service_settings', 'settings') }} page.
  2. Select Manage on the Text message sender IDs row.
  3. Select Change or Add text message sender ID.

If you want to receive text messages, your sender ID will be an 11-digit phone number.
