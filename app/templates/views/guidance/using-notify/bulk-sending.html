Bulk sending

You can send a batch of up to {{ max_spreadsheet_rows|format_thousands }} messages at once.

There’s a default daily limit of {{ rate_limits|formatted_list(before_each="", after_each="") }}. If you need to discuss these limits, contact us.

To send a batch of messages:

  1. Go to the {{ service_link(current_service, 'main.choose_template', 'templates') }} page.
  2. Add a new template or choose an existing template and select Get ready to send.
  3. Select Upload a list of email addresses, Upload a list of phone numbers, or Upload a list of addresses.

Find out how to personalise your messages.

You can also schedule a batch of emails and text messages.

International text messages

There’s a maximum international daily limit of 500 text messages. This limit is part of the daily text message allowance.

To change the international daily text message limit:

  1. Go to Settings
  2. Select Change on the International text message limit row

When changing the limit, you should think about:

  • how many international messages your service needs to send
  • any costs to your service if there is fraud, such as artificial inflation of traffic
