Emails use the default sans-serif typeface installed on the recipient’s device.

Our first choice is Helvetica, followed by Arial.

The default font size for emails is 19 pixels.

Recipients can change the font size in their device settings.

Emails support any Unicode characters.

You cannot change the typeface or font size.
